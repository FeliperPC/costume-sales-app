# Improved Requirement Description

I need to implement dynamic input masks for a form with the following fields:

Phone Number

Default mask: (99) 99999-9999

The mask must adjust dynamically based on user input:

Support both 10-digit and 11-digit Brazilian phone numbers

Example:

(99) 9999-9999 (landline)

(99) 99999-9999 (mobile)

📮 CEP (Postal Code)

Default mask: 99999-999

The mask should automatically format as the user types

📏 Measurements (Sizes)

Inputs must accept decimal values using a comma , as the decimal separator

Examples:

10,5

2,75

Only numeric values and a single comma should be allowed

⚙️ Functional Requirements

Masks must be applied in real time as the user types

Inputs should prevent invalid characters

Behavior should be user-friendly (no breaking cursor position while typing)

Should work well with copy/paste input
